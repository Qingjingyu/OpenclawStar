# Skill Packs

This file defines the three-layer skill structure for OpenclawStar.

## 1. Birth Pack
This is what every newborn claw should have on day 0.

Goal:
- talk
- search
- remember
- execute

Current birth pack:
- `search`
- `multi-search-engine`
- `agent-browser`
- `wechat-article-search`
- `baoyu-url-to-markdown`
- `find-skills`
- `ontology`
- `proactive-agent`
- `self-improvement`
- `skill-creator`
- `agent-team-orchestration`
- `actionbook`
- `agent-docs`
- `automation-workflows`
- `security-audit`
- `trust-verifier`
- `skill-vetter`
- `slides`
- `spreadsheets`

Rule:
- small, stable, general-purpose only
- no heavy role identity required
- no industry specialization required

## 2. Common Enhancement Pack
These are high-frequency add-ons for a general AI assistant.
They are not birth requirements, but they are worth curating in the repo.

Current common enhancement areas:
- `content`
- `research`
- `connectors`
- `memory`
- `dev`
- `ops`
- `design`
- `documents`
- `evaluation`

Rule:
- install when the claw starts to take on repeated real work
- do not auto-install everything blindly

## 3. Industry Pack
These are domain-specific skills.

Current industry pack:
- `finance`
  - `us-stock-analysis`

Future examples:
- recruiting
- legal
- healthcare
- education
- energy

Rule:
- install only when the claw has a clear domain mission
- keep industry packs independent from the birth pack

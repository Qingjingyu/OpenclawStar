# Skills

This folder is the curated skill library for OpenclawStar.

Principles:
- the newborn claw starts as a general AI assistant
- skills are organized by capability, not by company title
- this repo keeps the recommended starter skills in one place
- if a skill is not here, the claw can still search other sources

Use `skills/INDEX.md` first.

Current broad groups:
- search
- content
- research
- connectors
- growth
- coordination
- automation
- governance
- system
- finance

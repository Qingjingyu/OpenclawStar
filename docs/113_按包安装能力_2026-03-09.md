# 113 按包安装能力 2026-03-09

## 本次目标
让 OpenclawStar 不只是“装默认出生包”，还能够按能力包一键加装。

## 新能力
现在支持：
```bash
bash install.sh --pack content,research,dev
```

也支持同时加行业包：
```bash
bash install.sh --pack content,research,dev --finance
```

## 为什么重要
以前安装增强能力时，更多像手工拼装。
现在开始变成：
- 先出生
- 再按包加能力

这样建新虾会更快，也更不容易乱。

## 本次实现
1. `bootstrap/skill_packs.sh`
   - 记录每个 pack 里有哪些 skill
2. `bootstrap/install_skills_lib.sh`
   - 统一 skill 安装逻辑
3. `bootstrap/install_pack_skills.sh`
   - 专门安装指定 pack
4. `install.sh`
   - 新增 `--pack` 参数支持

## 当前可以直接用的 pack
- `content`
- `research`
- `connectors`
- `memory`
- `dev`
- `ops`
- `design`
- `documents`
- `evaluation`
- `finance`

## 例子
### 轻研究型助理
```bash
bash install.sh --pack content,research
```

### 开发型助理
```bash
bash install.sh --pack dev,evaluation
```

### 全能内容研究型助理
```bash
bash install.sh --pack content,research,documents --finance
```

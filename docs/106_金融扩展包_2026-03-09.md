# 106 金融扩展包 2026-03-09

## 本次决定
把 `us-stock-analysis` 归类为“金融扩展包”，不进入所有新虾的默认出生包。

## 为什么这样做
默认出生包只负责让新虾一出生就能：
- 聊天
- 搜索
- 记忆
- 执行

`us-stock-analysis` 很有用，但它属于专业能力，不是所有新虾都必须自带的通用底座。

## 这次落地了什么
1. 新增安装脚本：`Yoyoo/project/bootstrap/install_finance_skills.sh`
2. 在基线文档里明确“默认出生包”和“专业扩展包”分层
3. 目前金融扩展包内第一项是：`us-stock-analysis`

## 设计原则
- 默认新虾要轻，不要臃肿
- 专业能力按任务后装
- 一个行业一套扩展包，而不是全塞进默认模板

### 默认出生包
- OpenClaw core
- 模型
- 飞书
- QMD
- 默认推荐 skills
- 文件发送规则

### 金融扩展包

## 后续建议

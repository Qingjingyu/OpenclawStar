# 111 技能三层总清单 2026-03-09

## 目标
把 OpenclawStar 里的 skill 体系正式定型，避免以后越装越乱。

## 三层结构
### 第一层：出生包
这是所有新虾出生就应该有的能力。

标准：
- 通用
- 稳定
- 高频
- 低依赖

核心目标：
- 会聊天
- 会搜索
- 会记忆
- 会执行

### 第二层：常用增强包
这是通用 AI 助手开始干更多事情时，会经常加装的一层。

目前包括：
- content
- research
- connectors
- memory
- dev
- ops
- design
- documents
- evaluation

### 第三层：行业包
这是带明确职业目标、行业目标时才装的一层。

当前已落地：
- finance

## 这套结构的意义
以后新建一只虾时，不需要先想它是不是 CEO、CTO、工程师。
先问三件事：
1. 它有没有出生包？
2. 它需要哪些常用增强包？
3. 它有没有明确行业任务？

这样路径就清楚了。

## 结论
OpenclawStar 以后就是：
- 一个稳定的孵化仓库
- 一个分层清楚的能力仓库
- 一个可以持续演进的技能目录

# 109 第二批常用技能分类 2026-03-09

## 本次目标
继续把常用 skill 从“零散到处放”变成“仓库里可分类、可查找、可复用”的能力库。

## 这次新增的三类
### 1. content
这一类是内容抓取和内容速读。
适合：
- 看 X / YouTube / B站内容
- 快速提炼视频和帖子信息

### 2. research
这一类是研究检索和网页抓取。
适合：
- 全网搜信息
- 抓复杂网页
- 看 Reddit 讨论
- 做调研

### 3. connectors
这一类不是直接产出内容，而是帮龙虾接更多平台。
适合：
- 需要打通更多平台渠道时使用
- 比如 X、Reddit、YouTube、GitHub、小红书等

### connectors
- `agent-reach`

## 原则
- 默认出生包只放最基础、最稳的能力
- 第二批常用 skill 放进策展仓库，但不默认全装
- 需要时再按任务装

## 意义
这样以后你建一只新虾时，会有三层选择：
1. 出生包
2. 常用扩展包
3. 专业行业包

路径就清楚了，不会再变成一堆 skill 混在一起。

# 114 OpenClaw 学习地图 2026-03-09

## 目标
把常看的 OpenClaw 相关资料按层分好，避免以后学习路线混乱。

## 第一层：官方主干（最高优先级）
这层决定 OpenClaw 现在到底是什么、官方推荐怎么安装、怎么运行、怎么扩展。

### 1. 主体官网
- `https://openclaw.ai/`
- 用途：理解 OpenClaw 对外主张、核心定位、生态入口。

### 2. 官方文档
- `https://docs.openclaw.ai/start/getting-started`
- 用途：安装、启动、配置、上手。

### 3. 官方 GitHub 代码仓库
- `https://github.com/openclaw/openclaw`
- 用途：看 README、版本变化、真实代码结构、issue 和 release 节奏。

### 4. 官方技能市场 / 技能入口
- `https://clawhub.ai/`
- 用途：看技能生态、找 skill、理解技能分布。

## 第二层：技能生态（第二优先级）
这层决定“现在外面有哪些 skill 值得看、值得装、值得收进我们的仓库”。

### 1. Awesome OpenClaw Skills
- `https://github.com/VoltAgent/awesome-openclaw-skills`
- 用途：大索引、大盘点，适合找方向、看分类、看高频能力。

### 2. Skills.sh
- `https://skills.sh/`
- 用途：更广义的 agent skill 目录，不只限 OpenClaw，适合补充灵感。

### 3. 单点技能仓库示例
- `https://github.com/Jane-xiaoer/x-fetcher`
- 用途：专项能力补丁，适合单技能深挖。

## 第三层：案例生态（第三优先级）
这层决定“别人已经把 OpenClaw 用成了什么样”，适合拿来找产品方向和真实场景。

### 1. Awesome OpenClaw Use Cases
- `https://github.com/hesamsheikh/awesome-openclaw-usecases`
- 用途：看真实使用案例，帮助判断什么能力最值钱。

### 2. BotLearn
- `https://botlearn.ai/zh`
- 用途：更像“AI Agent 入学路径”，适合看非技术用户如何理解和学习。

### 3. docs-openclaw
- `https://github.com/Zolobaby/docs-openclaw`
- 用途：中文资料补充。

### 4. Claw123
- `https://claw123.ai/`
- 用途：技能导航和中文生态参考。

### 5. AI OpenClaw Skeletons
- `https://github.com/1596941391qq/ai-openclaw-skeletons`
- 用途：数字员工骨架思路，适合拿来对比我们自己的 OpenclawStar 设计。

## 第四层：方法论与外围能力仓库（吸收，不照抄）
这层不是 OpenClaw 官方主干，但对我们做能力仓库、开发规范、设计能力非常有帮助。

### 值得长期参考
- `https://github.com/affaan-m/everything-claude-code`
- `https://github.com/obra/superpowers`
- `https://github.com/nextlevelbuilder/ui-ux-pro-max-skill`
- `https://github.com/daymade/claude-code-skills`
- `https://github.com/code-yeongyu/oh-my-opencode`

用途：
- 学技能体系怎么组织
- 学开发规范怎么封装成 skill
- 学 UI/UX、研究、质量、流程类能力怎么整理

## 第五层：进化实验层（先研究，后接入）
这一层不是默认新虾出生包的一部分，更适合单独研究和实验。

### EvoMap / Evolver
- `https://evomap.ai/skill.md`
- `https://evomap.ai/wiki`
- `https://evomap.ai/blog`
- `https://github.com/EvoMap/evolver`

用途：
- 学“AI 自进化”“演化协议”“长期协作”的思路
- 不直接等于普通 skill 仓库
- 目前更适合作为实验线，不适合直接混进 OpenclawStar 的基础孵化流程

## 我们当前的学习顺序建议
### 第一优先级
- 官方官网
- 官方文档
- 官方 GitHub
- ClawHub

### 第二优先级
- Awesome OpenClaw Skills
- Skills.sh
- 单技能仓库（如 x-fetcher）

### 第三优先级
- Awesome Use Cases
- BotLearn
- 中文资料站

### 第四优先级
- everything-claude-code
- superpowers
- UI/UX 和外围 skill 仓库

### 第五优先级
- EvoMap / Evolver

## 对 OpenclawStar 的意义
OpenclawStar 以后继续做三件事：
1. 孵化新虾
2. 维护 curated skill 仓库
3. 按外部生态持续吸收高价值能力

也就是说：
- 官方资料负责校准方向
- 技能生态负责补能力
- 案例生态负责反推需求
- 进化层负责未来实验
